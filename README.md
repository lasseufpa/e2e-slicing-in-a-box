# e2e-slicing-in-a-box

# Introduction

# Requirements
Tested with **Ubuntu 18.04.** / **Ubuntu 20.04**

# Dependencies:
1. ContainerNet
2. Ryu
3. NS-3 mmWave
4. UERANSIM
5. python3 & pip
6. Docker
